## LikeLion 14TH FrontEnd Assignment <br> 경북대학교 멋쟁이 사자처럼 14기 프론트엔드 과제

### 사용법

1. 해당 Repository 를 Local 에 Clone 해주세요

```bash
git clone https://github.com/LikeLion-KNU/FE_likelion_Study_team1_14th.git
```

2. main 브랜치에서 본인의 영문이름_과제날짜 로 된 브랜치를 파고 checkout 해주세요

```bash
git checkout -b kimjihoon_0401
```

3. 해당 날짜 폴더 (ex. FE_0401) 하위에 **❗️영문이름으로 된 디렉토리를 만들고❗️** 과제 코드를 작성해주세요

```bash
cd FE_0401
mkdir kimjihoon
cd kimjihoon
```

6. 해당 브랜치에서 작업 후 변경사항을 커밋, 푸시 해주세요

```bash
git add .
git commit -m "김지훈_0401 멋사 1주차 과제"
git push origin kimjihoon_0401
```

6. Github 에서 브랜치에 대한 Pull Requests 를 올려주세요
